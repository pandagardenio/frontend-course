As you already know, divs are the block elements by default.
As we can see, when an element has display: block; elements do not start a new line.
As we can see, when an element has display: block; margins, padding, width and height are correctly applied.
As you already know, we can assign display: block; to any element. In this case to a span.
As you already know, spans are inline elements by default. As we can seee, when an element has display: inline; it does not start a new line. As we can see, when an element has display: inline; margins are correctly applied, but padding is applied in an incorrect way and widht and height is completely ignored.

As we can see here, elements with display: inline; is very useful to give styles inside text for example.

However, when we apply display: inline-block; we can see that margins, padding, width and height are correctly applied. As we can seee, when an element has display: inline-block it does not start a new line.
