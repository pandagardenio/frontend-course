.flex--basic { display: flex; }

As we can see, when we add display flex to a container, their children are laid out following the main axis (x-axis by default) even if they have display block.

Only with display: flex;, children elements get squeezed in case their total width is bigger than their parent's container width.

.flex--wrap { display: flex; flex-wrap: wrap; }

Adding the flex-wrap: wrap; property to the container, we tell CSS to put wrap the line in case children total width is bigger than their parent's container width.

flexbox allows us to make layouts in a very easy way.
